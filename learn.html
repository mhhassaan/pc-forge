CPU (Processor)

The brain of your computer, responsible for executing instructions and performing calculations.

What is a CPU?

The Central Processing Unit (CPU) is the primary component that executes instructions from programs. It performs calculations, logic operations, and controls the flow of data through the computer.

Key Specifications

  • Cores: Number of processing units (more cores = better multitasking)
  • Threads: Number of simultaneous tasks (usually 2x cores)
  • Clock Speed: Measured in GHz, determines how fast it processes
  • TDP: Thermal Design Power, indicates heat output and power consumption

Popular Brands

Intel and AMD are the two main CPU manufacturers. AMD Ryzen offers excellent value, while Intel Core provides strong single-thread performance.

GPU (Graphics Card)

Handles graphics rendering and parallel processing tasks for gaming, video editing, and AI.

What is a GPU?

The Graphics Processing Unit (GPU) specializes in rendering images and videos. Modern GPUs are also used for machine learning, cryptocurrency mining, and scientific computing.

Key Specifications

  • VRAM: Video memory (8GB, 12GB, 24GB, etc.)
  • CUDA Cores/Stream Processors: Parallel processing units
  • Memory Bandwidth: Speed of data transfer
  • TDP: Power consumption and heat output

Popular Brands

NVIDIA dominates with their GeForce RTX series, while AMD offers competitive Radeon RX cards. Intel Arc is emerging as a new player.

Motherboard

The main circuit board that connects all components and allows them to communicate.

What is a Motherboard?

The motherboard is the central hub that connects the CPU, GPU, RAM, storage, and all other components. It provides power distribution and enables communication between parts.

Key Specifications

  • Socket Type: Determines CPU compatibility (AM5, LGA1700, etc.)
  • Form Factor: Size (ATX, Micro-ATX, Mini-ITX)
  • PCIe Version: Affects GPU and storage speed (PCIe 4.0, 5.0)
  • RAM Slots: Number of memory modules supported

Popular Brands

ASUS, MSI, Gigabyte, and ASRock are leading manufacturers offering boards for different budgets and use cases.

RAM (Memory)

Temporary storage for active programs and data, enabling fast access during computation.

What is RAM?

Random Access Memory (RAM) is volatile memory that temporarily stores data and programs currently in use. It's much faster than storage but loses all data when powered off.

Key Specifications

  • Capacity: Measured in GB (16GB, 32GB, 64GB)
  • Speed: Measured in MHz (5600MHz, 6000MHz)
  • Type: DDR4 or DDR5 (newer is faster)
  • Latency: CAS latency affects responsiveness

Recommendations

16GB is standard for gaming and productivity. 32GB for content creation and multitasking. 64GB+ for professional workloads.

Storage (SSD/HDD)

Permanent storage for your operating system, programs, and files.

What is Storage?

Storage devices permanently hold your data. SSDs (Solid State Drives) are faster and more reliable than HDDs (Hard Disk Drives), which use spinning platters.

Key Specifications

  • Capacity: 500GB to 4TB+ for different needs
  • Type: NVMe SSD (fastest), SATA SSD, or HDD
  • Speed: Read/write speeds in MB/s
  • Interface: M.2, 2.5", 3.5" form factors

Recommendations

Use NVMe SSD for OS and programs (500GB-1TB). Add secondary SATA SSD or HDD for bulk storage.

Power Supply

Converts wall power to usable electricity for all components.

What is a Power Supply?

The Power Supply Unit (PSU) converts AC power from your wall outlet to DC power needed by computer components. Quality matters for stability and longevity.

Key Specifications

  • Wattage: Total power output (650W, 850W, 1000W)
  • Efficiency Rating: 80+ Bronze, Gold, Platinum
  • Modular: Fully modular cables reduce clutter
  • Protection: Over-current, over-voltage protection

Sizing Guide

Calculate total system power draw and add 20-30% headroom. A 750W PSU handles most gaming builds.
